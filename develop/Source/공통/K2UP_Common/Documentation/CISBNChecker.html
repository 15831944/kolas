static INT ConvertISBN(CHAR* szUnFormattedISBN, CHAR* szCorrectFormattedISBN, INT* nCorrect)
[static] ISBN을 규칙에 의거하여 변환한다.
ISBN에 '()'가 있으면(Set ISBN인 경우) '()'를 떼서 ISBN 체크를 하며,
'-' 위치도 정확하게 변경한다.
'89'로 시작하는 국내 ISBN에 대해서만 처리한다.

ECO_ConvertISBN(in, out, &nreturn );
if ( nreturn ) {
if (nreturn == -1)
MessageBox( "ISBN 자리수를 확인하여 주십시요.", "ISBN CHECK", MB_ICONEXCLAMATION);
if (nreturn == -2)
MessageBox( "ISBN 체크디지트를 확인하여 주십시요.", "ISBN CHECK", MB_ICONEXCLAMATION);
...
}

Parameters:
szUnFormattedISBN - 입력ISBN
szCorrectFormattedISBN - ISBN에 맞게 변형되어진 출력값( 89-XXX-XXXX-X )
nCorrect - 입력ISBN이 자리수가 안맞으면 -1, 체크디지트가 틀리면 -2 맞으면 0

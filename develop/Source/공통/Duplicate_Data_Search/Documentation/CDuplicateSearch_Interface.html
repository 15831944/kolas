Documentation

사용자가 실제 중복/복본조사를 수행할 수 있는 메소드를 포함 메소드의 종류는 일괄 중복/복본조사 서명복본조사로 구성되어진다.
일괄 복본/중복 조사는 쓰레드로 동작하며 중복/복본 데이터 구축과 이를 출력 하는 루틴이 분리되어 있다.
서명 복본조사의 경우 쓰레드로 동작하지 않으며 역시 데이터구축과 결과출력 루틴이 분리되어 있다.
일괄 중복/복본조사의 쓰레드 수행과 이의 자료처리상태 및 데이터 구축 동작을 제어하기 위한 CDataProcessingDlg 와의 상호작용을 위하여 본 클래스는 CWin클래스를 상속받으며 일괄 복본/중복 조사를 수행하는 쪽에서 CWin클래스의 Create메소드를 호출하여 윈도우 Handle를 생성하고 CDataProcessingDlg클래스에 정의된 JOB_CANCEL 사용자 메시지로서 쓰레드의 동작을 제어한다.

Send Mail Manager Class 메일 매니저.
1. Registry에서 시스템에 설정된 메일 서버와 이메일 주소를 가져옴.(GetDefaultMailServerInfo)
1. 또는 메일 서버와 이메일 주소를 직접 설정.(SetDefaultMailServer)
2. 보낼 메일 레코드를 추가.(AddMailRecord)
3. Dialog List Control에서 선택된(checked) 메일 전송.( SendMail(TRUE,TRUE) )
3. 또는 Background 모드로 메일 전송.( SendMail(FALSE,FALSE) )
4. 메일 전송여부 확인.(GetCountMailSended, GetCountMailFailed)
* 메일전송에 대한 로그파일 : ..../tunning/(YYYYMMDD)SmtpLog.txt

nFileAttachCnt - 메일 첨부파일 개수
